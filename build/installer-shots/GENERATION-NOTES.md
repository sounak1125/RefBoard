# RefBoard installer cinematic

Redesigned from scratch on 2026-07-18 as an original five-scene launch-film sequence. The final frames are 2560 x 1440 PNGs and intentionally keep the left third dark and quiet for installer copy and controls.

These images do not reproduce RefBoard screenshots or another company's installer. They use RefBoard's own visual language: a dotted infinite canvas, floating reference planes, precise blue selection geometry, drawn annotations, timeline ribbons, and the RefBoard mark.

## Sequence

1. `final/01-build-your-visual-world.png` — an infinite field of references connects into a visual world.
2. `final/02-focus-on-every-detail.png` — one reference resolves into precise focus and alignment.
3. `final/03-think-directly-on-the-canvas.png` — luminous annotation layers turn observation into thought.
4. `final/04-turn-references-into-motion.png` — reference planes unfold into timeline and audio ribbons.
5. `final/05-ready-for-your-next-workflow.png` — the journey resolves into the RefBoard identity.

## Art direction

- Premium cinematic 3D motion-design keyframes rather than literal UI screenshots.
- Near-black charcoal and navy space with restrained cobalt-blue light and sparse warm amber accents.
- Smoked glass, anodized-black material, fine dotted depth fields, and minimal bloom.
- No fake software chrome, borrowed branding, device mockups, readable generated text, or people.
- Strong silhouettes and low visual noise so the frames remain readable in a compact installer window.

## Generation method

Scenes 1–4 were generated independently with the built-in image-generation path using a shared visual specification. Scene 5 used `build/icon-master.png` only as a shape reference so the dimensional end mark remains recognizably RefBoard. The selected outputs were normalized locally to 2560 x 1440 with high-quality Lanczos scaling.
